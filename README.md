# Gulp-Starter-Kit

### Installation
First install [Node.js] [node.js]

Then, you need to install Gulp globally:

```sh
$ npm i -g gulp
```

Install npm dependecies:

```sh
$ npm install
```

Get Gulp running:

```sh
$ gulp
```
